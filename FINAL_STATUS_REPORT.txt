   Check: python -m pip list | grep Flask

Q: Frontend buttons not working?
A: Verify backend is running
   Check browser console for errors (F12)
   Ensure localStorage is enabled

Q: Database not saving data?
A: Check file permissions on d:\Project1\
   Verify health_assistant.db exists
   Run: python check_database.py

   Run: python test_auth_flow.py
